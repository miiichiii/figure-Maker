以下をそのまま埋めて送ってください。

```
[タイトル]
- 例: SVG貼り付けでベクターにならない

[発生日時]
- YYYY-MM-DD HH:MM

[Error ID]
- 画面のエラー通知に表示されたID

[再現手順]
1.
2.
3.

[期待結果]
-

[実際の結果]
-

[Paste report / history]
- Copy report または Copy history の内容を貼る

[環境]
- OS:
- ブラウザ:
- バージョン:

[補足]
- スクリーンショット、入力データ、対象SVG など
```

## デバッグモード
- 通常: `index.html`
- 詳細ログ: `index.html?debug=1`

`debug=1` のとき、エラー出力に詳細フィールド（event history など）を含みます。
