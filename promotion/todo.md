# Figure Maker プロモーション TODO

## アプリの売りポイント（抽出済み）

### コアバリュー
- **完全無料** — 利用料ゼロ
- **ログイン不要** — アカウント作成なし、すぐ使える
- **ブラウザのみで完結** — インストール不要、どのOSからでもアクセス可能
- **論文Figure特化** — Illustratorの複雑な機能を削ぎ落とし、研究者が本当に必要な機能だけを搭載

### ファイル互換性
- **Adobe Illustrator (.ai) ファイルをドラッグ&ドロップで取り込み可能**
- **SVGファイルをネイティブに扱う** — 開く / 保存 / インポート
- **PDF書き出し対応** — 論文投稿用PDFをワンクリック生成
- **画像ペースト対応** — クリップボードから直接画像を貼り付け（TIFF対応含む）

### 強力な編集ツール
- **ダイレクト選択ツール（White Arrow）** — グループ化を無視して個別要素を直接選択・編集
- **選択ツール（Black Arrow）** — 通常のオブジェクト選択
- **ハンドツール** — Space長押しでキャンバスパン
- **クロップツール** — 画像の非破壊トリミング
- **整列・分布ツール** — 左/中央/右/上/中/下揃え、水平・垂直等間隔分布
- **スマートガイド** — 移動・リサイズ時のスナップアシスト
- **回転** — 選択枠の外をドラッグで自由回転（Shift=15°スナップ）

### 図形とテキスト
- 矩形、楕円、直線、三角形、テキストを追加
- **矢印変換（Apply Arrow）** — 線をワンクリックで矢印に変換
- **マークラベル（A-Z）** — 右クリックメニューからFigureパネル用ラベルを付与
- **Figure Title / Figure Number** — Supplementary Figure対応
- ストローク色・塗りつぶし色・ストローク幅のカスタマイズ
- テキストの太字・フォントサイズ変更

### データ保全
- **自動保存（Time Machine機能）** — 1分 / 10分 / 30分間隔で自動スナップショット
- **IndexedDBベース** — 大容量画像もブラウザ内に安全に保存
- **スナップショットリスト** — 過去の状態を一覧から復元可能
- **Undo/Redo** — 最大30段階の履歴管理

### その他
- **キーボードショートカット充実** — Illustrator風の操作体系（V, A, H, R, E, L, T, Q等）
- **ズーム** — スライダー + Ctrl/Cmd+ホイール + ピンチ対応
- **A4プリセット** — Portrait/Landscapeワンクリック切替
- **グループ化/グループ解除** — Ctrl+G等
- **コンテキストメニュー** — 右クリックでCopy/Cut/Paste/Mark/Group/Duplicate
- **エラーログ機能** — 問題発生時にログをダウンロード可能
